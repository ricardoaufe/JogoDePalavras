Introdução Nós Jogo

Sobre o Jogo

Jogo de Palavras é um jogo cujo objetivo é formar o máximo de palavras possíveis dentro de um tempo limite!

Detalhes sobre o Jogo

Ao clicar no botão Jogar, o jogador terá a opção de escolher entre 3 modos. Fácil, médio e difícil, cada qual exigindo mais velocidade e coordenação.

Como jogar?

É simples!, após selecionar o modo, um timer se ativará e você terá 60 segundos para poder clicar nas letras que caem sem parar para poder formar palavras. Ao formar uma palavra, aperte ENTER para conferir se a palavra existe. Caso tenha clicado em alguma letra sem querer, aperte o botão BACKSPACE para apagar o útlimo caractere!
